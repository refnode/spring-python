Yaml configuration format for SpringPython ApplicationContext
=============================================================

Quickstart
==========

* git clone git@github.com:snwilhelm/springpython.config.yaml.git
* cd springpython.config.yaml
* python bootstrap.py
* bin/buildout

Alternative:

* git clone git@github.com:snwilhelm/springpython.config.yaml.git
* cd springpython.config.yaml
* make bootstrap2.7
